# API Documentation

## Авторизация

Все запросы требуют заголовок:

```
Authorization: Token <your_token>
```

Токен можно создать в профиле пользователя

---

## User

### GET /user/me

Возвращает профиль авторизованного пользователя, включая баланс, уровень и лимиты подписок.

**Rate limit:** 4 запроса в секунду

**Headers**

| Название        | Обязательный | Описание          |
|-----------------|:------------:|-------------------|
| `Authorization` | да           | `Token <token>`   |

**Response `200`**

```json
{
  "telegramId": "123456789",
  "username": "john_doe",
  "level": 3,
  "volume": 650.0,
  "tonBalance": 12.45,
  "subscriptionLimit": 5,
  "isAutobuyEnabled": true,
  "hideInLeaderboard": true,
  "isBan": false,
  "ref": null,
  "lolPopsStaked": 0,
  "ionicDryersStaked": 0,
  "levelConfig": {
    "level": 3,
    "volume": 500,
    "cashback": 0.0125,
    "adminFee": 0.26,
    "leaderboardFee": 0.05,
    "refFee": 0.08,
    "stakingFee": 0.03,
    "subRefFee": 0.02,
    "feeSum": 0.44
  }
}
```

**Поля ответа**

| Поле                    | Тип       | Описание                                                                 |
|-------------------------|-----------|--------------------------------------------------------------------------|
| `telegramId`            | `string`  | Telegram ID пользователя                                                 |
| `username`              | `string`  | Telegram username                                                        |
| `level`                 | `number`  | Уровень пользователя (1–9)                                               |
| `volume`                | `number`  | Торговый объём в TON                                                     |
| `tonBalance`            | `number`  | Баланс в TON (округлён до 2 знаков)                                      |
| `subscriptionLimit`     | `number`  | Итоговый лимит подписок с учётом бонусов                                 |
| `isAutobuyEnabled`      | `boolean` | Включён ли автопокупатель                                                |
| `hideInLeaderboard`     | `boolean` | Скрыт ли в таблице лидеров                                               |
| `lolPopsStaked`         | `number`  | Количество застейканных LolPops (+1 к лимиту за каждый)                  |
| `ionicDryersStaked`     | `number`  | Количество застейканных Ionic Dryers (+10 к лимиту за каждый)            |
| `ref`                   | `string\|null` | Реферальный ID того, кто пригласил                                  |
| `levelConfig`           | `object`  | Конфиг текущего уровня (комиссии, кэшбэк — см. ниже)                    |

**Поля `levelConfig`**

| Поле             | Тип      | Описание                              |
|------------------|----------|---------------------------------------|
| `level`          | `number` | Номер уровня                          |
| `volume`         | `number` | Минимальный объём для достижения уровня |
| `cashback`       | `number` | Кэшбэк с покупок (доля, напр. `0.0125` = 1.25%) |
| `adminFee`       | `number` | Комиссия платформы                    |
| `leaderboardFee` | `number` | Отчисление в пул лидерборда           |
| `refFee`         | `number` | Отчисление рефереру                   |
| `subRefFee`      | `number` | Отчисление суб-рефереру               |
| `stakingFee`     | `number` | Отчисление в стейкинг-пул             |
| `feeSum`         | `number` | Суммарная комиссия                    |

**Ошибки**

| Код  | Описание                                      |
|------|-----------------------------------------------|
| `401` | Токен отсутствует или недействителен         |
| `403` | Пользователь не найден или заблокирован       |
| `429` | Превышен rate limit (4 req/s)                 |

### GET /user/subscriptions

Возвращает список подписок авторизованного пользователя.

**Rate limit:** 4 запроса в секунду

**Headers**

| Название        | Обязательный | Описание        |
|-----------------|:------------:|-----------------|
| `Authorization` | да           | `Token <token>` |

**Response `200`**

```json
[
  {
    "_id": "664a1f2e3c4b5a6d7e8f9012",
    "subscriptionName": "My Pepe alerts",
    "collectionName": "PlushPepe",
    "modelNames": ["Sad Pepe"],
    "backdropNames": ["Forest"],
    "numberPattern": "X888",
    "portalsNotifyMaxPrice": 10.0,
    "notifyTg": true,
    "notifyPortals": false,
    "notifyTonnel": false,
    "notifyMrkt": true,
    "notifyGetgems": false,
    "portalsAutobuy": true,
    "portalsAutobuyMaxPrice": 8.5,
    "portalsAutobuyQuantity": 3,
    "autobuyTg": true,
    "autobuyPortals": true,
    "autobuyTonnel": false,
    "autobuyMrkt": false,
  }
]
```

**Поля объекта подписки**

| Поле                      | Тип               | Описание                                                                             |
|---------------------------|-------------------|--------------------------------------------------------------------------------------|
| `_id`                     | `string`          | ID подписки                                                                          |
| `subscriptionName`        | `string`          | Название подписки                                                                    |
| `collectionName`          | `string`          | Коллекция, за которой следит подписка. `all-collections` — все коллекции             |
| `modelNames`              | `string[]`        | Фильтр по моделям (пустой массив = все)                                              |
| `backdropNames`           | `string[]`        | Фильтр по фонам (пустой массив = все)                                                |
| `numberPattern`           | `string\|null`    | Паттерн фильтрации по номеру подарка (макс. 6 символов). `null` — без фильтра        |
| `portalsNotifyMaxPrice`   | `number\|null`    | **Легаси-название.** Максимальная цена для уведомлений                               |
| `notifyTg`                | `boolean`         | Уведомлять при появлении подарка в Telegram Market                                   |
| `notifyPortals`           | `boolean`         | Уведомлять при появлении подарка на Portals                                          |
| `notifyTonnel`            | `boolean`         | Уведомлять при появлении подарка на Tonnel                                           |
| `notifyMrkt`              | `boolean`         | Уведомлять при появлении подарка на Mrkt                                             |
| `notifyGetgems`           | `boolean`         | Уведомлять при появлении подарка на Getgems                                          |
| `portalsAutobuy`          | `boolean`         | **Легаси-название.** Включён ли автобай в принципе (слово `portals` не несёт смысла) |
| `portalsAutobuyMaxPrice`  | `number\|null`    | **Легаси-название.** Максимальная цена для автобая                                   |
| `portalsAutobuyQuantity`  | `number`          | **Легаси-название.** Лимит автобаев по данной подписке                               |
| `autobuyTg`               | `boolean`         | Автобай в Telegram Market                                                            |
| `autobuyPortals`          | `boolean`         | Автобай в Portals                                                                    |
| `autobuyTonnel`           | `boolean`         | Автобай в Tonnel                                                                     |
| `autobuyMrkt`             | `boolean`         | Автобай в Mrkt                                                                       |

**Ошибки**

| Код   | Описание                                  |
|-------|-------------------------------------------|
| `401` | Токен отсутствует или недействителен      |
| `429` | Превышен rate limit (4 req/s)             |

### GET /user/transactions

Возвращает paginated список транзакций пользователя.

**Rate limit:** 4 запроса в секунду

**Headers**

| Название        | Обязательный | Описание        |
|-----------------|:------------:|-----------------|
| `Authorization` | да           | `Token <token>` |

**Query параметры**

| Параметр    | Тип      | По умолчанию | Описание                                                                |
|-------------|----------|:------------:|-------------------------------------------------------------------------|
| `page`      | `number` | `0`          | Номер страницы (0-based)                                                |
| `limit`     | `number` | `20`         | Размер страницы (макс. 100)                                             |
| `currency`  | `string` | `ton`        | Валюта: `ton`                                              |
| `type`      | `string` | —            | Фильтр по типу, можно несколько через запятую: `deposit,withdrawal`     |
| `minAmount` | `number` | —            | Минимальная сумма транзакции                                            |
| `maxAmount` | `number` | —            | Максимальная сумма транзакции                                           |

**Response `200`**

```json
{
  "items": [
    {
      "id": "664a1f2e3c4b5a6d7e8f9012",
      "type": "deposit",
      "amount": 10.5,
      "currency": "ton",
      "payload": "",
      "date": "2024-05-20T12:00:00.000Z"
    }
  ],
  "page": 0,
  "limit": 20,
  "totalPages": 3,
  "totalItems": 57
}
```

**Типы транзакций (`type`)**

| Значение                      | Описание                              |
|-------------------------------|---------------------------------------|
| `deposit`                     | Пополнение баланса                    |
| `withdrawal`                  | Вывод средств                         |
| `purchase`                    | Покупка подарка                       |
| `purchase_fee`                | Комиссия с покупки                    |
| `ref_percent`                 | Реферальное вознаграждение            |
| `sub_ref_percent`             | Суб-реферальное вознаграждение        |
| `ref_cashback`                | Кэшбэк                                |
| `leaderboard_fee`             | Отчисление в пул лидерборда           |
| `leaderboard_prize`           | Приз лидерборда                       |
| `staking_fee`                 | Отчисление в стейкинг-пул             |
| `staking_payout`              | Выплата из стейкинга                  |
| `subscription_limit_increase` | Увеличение лимита подписок            |
| `safe_deposit`                | Перевод в сейф                        |
| `safe_withdrawal`             | Вывод из сейфа                        |
| `stars_top_up`                | Пополнение через Telegram Stars       |
| `stars_refund`                | Возврат Stars                         |

**Ошибки**

| Код   | Описание                                          |
|-------|---------------------------------------------------|
| `400` | Невалидные параметры (page, minAmount > maxAmount)|
| `401` | Токен отсутствует или недействителен              |
| `429` | Превышен rate limit (4 req/s)                     |

### GET /user/stats

Возвращает агрегированную статистику покупок пользователя в TON.

**Rate limit:** 4 запроса в секунду

**Headers**

| Название        | Обязательный | Описание        |
|-----------------|:------------:|-----------------|
| `Authorization` | да           | `Token <token>` |

**Query параметры**

| Параметр    | Тип      | По умолчанию | Описание                                                        |
|-------------|----------|:------------:|-----------------------------------------------------------------|
| `startDate` | `string` | —            | Начало диапазона дат (ISO 8601, напр. `2024-01-01T00:00:00.000Z`) |
| `endDate`   | `string` | —            | Конец диапазона дат (ISO 8601)                                  |

**Response `200`**

```json
{
  "purchasesCount": 42,
  "purchasesVolume": 210.5,
  "holdVolume": 85.12,
  "profit": 14.37
}
```

**Поля ответа**

| Поле               | Тип      | Описание                                                                 |
|--------------------|----------|--------------------------------------------------------------------------|
| `purchasesCount`   | `number` | Количество покупок                                                       |
| `purchasesVolume`  | `number` | Суммарный объём покупок (цена + комиссия) в TON                          |
| `holdVolume`       | `number` | Объём непроданных подарков (округлён до 2 знаков)                        |
| `profit`           | `number` | Прибыль по проданным подаркам с учётом кэшбэка (округлена до 2 знаков)  |

**Ошибки**

| Код   | Описание                             |
|-------|--------------------------------------|
| `400` | Невалидный формат дат                |
| `401` | Токен отсутствует или недействителен |

### POST /user/subscribe

Создаёт подписку.

**Rate limit:** 1 запрос в 2 секунды

**Headers**

| Название       | Обязательный | Описание        |
|----------------|:------------:|-----------------|
| `Authorization`| да           | `Token <token>` |

**Body (JSON)**

```json
{
  "subscriptionName": "My Pepe alerts",
  "collectionName": "Plush Pepe",
  "modelNames": ["Sad Pepe"],
  "backdropNames": ["Forest"],
  "numberPattern": "X888",
  "portalsNotifyMaxPrice": 10.0,
  "notifyTg": true,
  "notifyPortals": false,
  "notifyTonnel": false,
  "notifyMrkt": true,
  "notifyGetgems": false,
  "portalsAutobuy": true,
  "portalsAutobuyMaxPrice": 8.5,
  "portalsAutobuyQuantity": 3,
  "autobuyTg": true,
  "autobuyPortals": true,
  "autobuyTonnel": false,
  "autobuyMrkt": false,
  "forwardToChat": "@mychannel",
  "forwardToTopic": "12345"
}
```

**Поля body**

| Поле                     | Тип             | Обязательный | Описание                                                                  |
|--------------------------|-----------------|:------------:|---------------------------------------------------------------------------|
| `subscriptionName`       | `string`        | да           | Название подписки (макс. 40 символов)                                     |
| `collectionName`         | `string`        | да           | Коллекция. Конкретное название или `all-collections` для всех коллекций   |
| `modelNames`             | `string[]`      | да           | Фильтр по моделям (пустой массив = все, макс. 100)                        |
| `backdropNames`          | `string[]`      | да           | Фильтр по фонам (пустой массив = все, макс. 100)                          |
| `numberPattern`          | `string\|null`  | нет          | Паттерн по номеру подарка (макс. 6 символов, только `[A-Za-z0-9]`). Цифры — точное совпадение, буквы — wildcard (`\d`), одинаковые буквы = одинаковые цифры. Примеры: `888` — точно 888, `X888` — ?888, `AAA` — 111/222.../999 |
| `portalsNotifyMaxPrice`  | `number\|null`  | нет          | **Легаси-название.** Максимальная цена для уведомлений                    |
| `notifyTg`               | `boolean`       | нет          | Уведомлять при появлении подарка на Fragment                              |
| `notifyPortals`          | `boolean`       | нет          | Уведомлять при появлении подарка на Portals                               |
| `notifyTonnel`           | `boolean`       | нет          | Уведомлять при появлении подарка на Tonnel                                |
| `notifyMrkt`             | `boolean`       | нет          | Уведомлять при появлении подарка на Mrkt                                  |
| `notifyGetgems`          | `boolean`       | нет          | Уведомлять при появлении подарка на Getgems                               |
| `portalsAutobuy`         | `boolean`       | нет          | **Легаси-название.** Включить автобай                                     |
| `portalsAutobuyMaxPrice` | `number`        | нет          | **Легаси-название.** Максимальная цена для автобая                        |
| `portalsAutobuyQuantity` | `integer`       | нет          | **Легаси-название.** Лимит автобаев по подписке (мин. 1 при автобае)     |
| `autobuyTg`              | `boolean`       | нет          | Автобай через Fragment                                                    |
| `autobuyPortals`         | `boolean`       | нет          | Автобай через Portals                                                     |
| `autobuyTonnel`          | `boolean`       | нет          | Автобай через Tonnel                                                      |
| `autobuyMrkt`            | `boolean`       | нет          | Автобай через Mrkt                                                        |
| `forwardToChat`          | `string`        | нет          | Username (`@channel`) или ID чата для пересылки уведомлений               |
| `forwardToTopic`         | `string`        | нет          | ID топика в чате (только цифры)                                           |

**Response `201`** — созданная подписка (см. поля в `GET /user/subscriptions`)

**Ошибки**

| Код   | Описание                                              |
|-------|-------------------------------------------------------|
| `400` | Невалидные параметры, лимит подписок достигнут, недостаточно баланса |
| `401` | Токен отсутствует или недействителен                  |
| `429` | Превышен rate limit (1 req / 2s)                      |

### POST /user/increase-limit

Увеличивает лимит подписок. Стоимость — **1 TON за единицу**. Списывается с баланса пользователя.

**Rate limit:** 1 запрос в 2 секунды

**Headers**

| Название        | Обязательный | Описание        |
|-----------------|:------------:|-----------------|
| `Authorization` | да           | `Token <token>` |

**Body (JSON)**

```json
{
  "amount": 3
}
```

| Поле     | Тип       | Обязательный | Описание                                     |
|----------|-----------|:------------:|----------------------------------------------|
| `amount` | `integer` | да           | Количество единиц лимита (целое, больше 0)   |

**Response `200`** — обновлённый объект пользователя (см. поля в `GET /user/me`)

**Ошибки**

| Код   | Описание                                    |
|-------|---------------------------------------------|
| `400` | `amount` не целое или ≤ 0, недостаточно TON |
| `401` | Токен отсутствует или недействителен        |
| `429` | Превышен rate limit (1 req / 2s)            |

**Rate limit:** 1 запрос в 2 секунды

**Headers**

| Название        | Обязательный | Описание        |

**Body (JSON)** — см. поля в `POST /user/subscribe`

**Response `200`** — пустой ответ

**Ошибки**

| Код   | Описание                                                              |
|-------|-----------------------------------------------------------------------|
| `400` | Невалидные параметры, недостаточно баланса                            |
| `401` | Токен отсутствует или недействителен                                  |
| `404` | Подписка не найдена или не принадлежит пользователю                   |
